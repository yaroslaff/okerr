Для okerr существует уже готовый набор проверок сервера. Чтобы удобнее ими пользоваться создадим файл конфигурации (/etc/okerr/okerrupdate) для okerrupdate командой: okerrmod -i {{textid}} --init. Запустите okerrmod без параметров чтобы установить несколько индикаторов: 

root@okerr:~# okerrmod 
okerr updated (200 OK) okerr:df-/@7rkd1xyyxo = 10.80
okerr updated (200 OK) okerr:df-/opt/okerr@7rkd1xyyxo = 10.80
okerr updated (200 OK) okerr:df-/root/repo@7rkd1xyyxo = 10.80
okerr updated (200 OK) okerr:la@7rkd1xyyxo = 0.74
okerr updated (200 OK) okerr:opentcp@7rkd1xyyxo = 22 25 80 4369 5355 5671 6379 8000 25672
Задача будет засчитана когда в проекте будет хотя бы один индикатор, установленный таким образом.

Дополнительная информация

  • okerrmod --init создает крон-файл /etc/cron.d/okerrmod чтобы обновлять индикаторы каждые 20 минут
  • Вы можете удалить предыдущие индикаторы, чтобы была возможность создать несколько новых индикаторов.
