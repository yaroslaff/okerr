Установите okerrclient на ваш linux сервер.
  • Убедитесь, что на вашем сервере установлены стандартные пакеты python и pip. Если их нет - установите (Debian/Ubuntu: sudo apt-get install python-pip)
  • Командой sudo pip3 install okerrclient установите пакет okerrclient
  • Создайте новый индикатор прямо с сервера, командой: okerrclient -i {{textid}} --name test:4 -s OK
