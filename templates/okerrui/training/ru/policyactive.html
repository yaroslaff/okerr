Правила, по которым okerr работает с индикаторами указываются в политиках. Политика - общая конфигурация, которая применяется ко всем индикаторам, которые ее используют. Каждый индикатор связан с одной политикой. Изначально в проекте есть политика Default (с периодом 1 час) и Daily (с периодом в 1 день).

На странице проекта добавьте политику с именем test:active. Настройте ее следующим образом:

  • Период проверки: один раз в 2 часа (2h)
  • Расписание перепроверок: 5 минут (5m).
  • Без перепроверок для восстановления (пустая строка)
  • Отключите автосоздание, прием обновлений через HTTP и SMTP
Создайте любой активный индикатор (Который исполняется сервером okerr, например, типа Ping или HTTP status code) и именем, начинающимся на 'test:', и назначьте ему эту политику.
