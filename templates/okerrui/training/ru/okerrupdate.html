Установите okerrclient на ваш linux сервер.
  • Убедитесь, что на вашем сервере установлены стандартные пакеты python3 и pip3. Если их нет - установите (Debian/Ubuntu: sudo apt install python3-pip)
  • Командой sudo pip3 install okerrupdate установите пакет okerrupdate
  • Создайте новый индикатор прямо с сервера, командой: okerrupdate -i {{textid}} test:4 OK
(Если вы увидите сообщение 'cannot read config' - все нормально, проигнорируйте его пока что)
