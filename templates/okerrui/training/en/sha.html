Watching news
Lets configure okerr to get notifications about changes on news site.
  • Create indicator (e.g., test:webpage) and set checkmethod 'HTTP SHA1 hash dynamic'.
  • Set url https://meduza.io/.
  • Wait until okerr will initialized field 'hash' (request retest).
Unlike usual (so-called 'static') indicators, this 'dynamic' indicator always will have 'OK' state. This is useful when any content of web page is good (does not requires any 'fix'), but you still will get alerts when content changes.
