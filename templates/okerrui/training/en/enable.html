In last task, okerrmod executed set of standard checks and updated (created) indicators. You can manage this set and use different sets of checks on different servers.

  • Run okerrmod --list to list all available checks. Checks with + sign are enabled.
  • Run sudo okerrmod --disable CHECK to disable any CHECK (e.g 'df')
  • Run sudo okerrmod --enable uptime to enable check "uptime"

Run okerrmod, it will perform only checks from template you created and assigned to server.
