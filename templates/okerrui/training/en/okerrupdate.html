Install okerrclient on your linux server.
  • Make sure your server has standard packages python and pip. If not - install it. (Debian/Ubuntu: sudo apt-get install python-pip)
  • Run command: sudo pip3 install okerrclient to install okerrclient.
  • Create indicator from server shell, with command okerrupdate -i {{textid}} test:4 OK
(If you will see 'cannot read config' error - thats OK, ignore it for now)
