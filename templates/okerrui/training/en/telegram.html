Connect Telegram messenger to your account.
  • Install Telegrap app on your phone.
  • (Recommended) Set name (username) for telegram account.
  • Set this name on your профиле.
  • Find user OkerrBot.
  • In chat OkerrBot make command /on
Command /on binds telegram chat (where you did this command) and account. Field 'Telegram username' of profile must match telegram username.
If you dont have username in telegram, you can either set username, or you will get error message with chat id. You can use this chat id instead of username for 'telegram username' field and repeat /on command.
More detailed instructions (in russian).
If you already linked your telegram account to profile, just click 'Check'.
After you linked account, you will get alerts in telegram and always can get brief status with command /sum. (Telegram has feature to mute chat if you dont want to get notifications, so you dont have to unlink it).
