Okerr has set of checks for server. To use it easily, create configuration file (/etc/okerr/okerrupdate) with command: okerrmod -i {{textid}} --init Run okerrmod (without any arguments) to update all indicators: 

root@okerr:~# okerrmod 
okerr updated (200 OK) okerr:df-/@7rkd1xyyxo = 10.80
okerr updated (200 OK) okerr:df-/opt/okerr@7rkd1xyyxo = 10.80
okerr updated (200 OK) okerr:df-/root/repo@7rkd1xyyxo = 10.80
okerr updated (200 OK) okerr:la@7rkd1xyyxo = 0.74
okerr updated (200 OK) okerr:opentcp@7rkd1xyyxo = 22 25 80 4369 5355 5671 6379 8000 25672
Task will be completed when project will have at least one indicator created this way.

Additional info

  • - okerrmod --init creates cron file /etc/cron.d/okerrmod, to update indicators every 20 minutes
  • - You may need to delete few old indicators to be able to create few more indicators.
