Okerr has set of safe and easy checks for server. To use it easily, install okerrclient configuration file: cp /usr/local/okerrclient/conf/okerrclient.conf /etc. Now, edit file /etc/okerrclient.conf and set text id. textid={{textid}} Run okerrclient (without any arguments) to update all indicators: 

root@braconnier:~# okerrclient 
okerr updated braconnier:uptime = 362949
okerr updated braconnier:la = 0.31
okerr updated braconnier:df-/ = 53.9
okerr updated braconnier:maxlogsz = 997513
okerr updated braconnier:opentcp = 22(sshd) 25(master) 80(apache2) 139(sm..
okerr updated braconnier:version = OK
okerr updated braconnier:df-inodes-/ = 7
okerr updated braconnier:okerr_errors = 0
...
Task will be completed when project will have at least one indicator created this way.

Bonus

Probably you want to make periodical updates via cron or as system services (details - via link below). You you will not make this, indicators will 'expire' after some time and switch to 'ERR' status. You will get alert about this via email or telegram.
