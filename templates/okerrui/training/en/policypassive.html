At last task we created policy, which are good for active check. Now we will create policy for passive checks.

At project page create policy 'test:passive' for passive indicators. Configure it this ways:

  • Set 'secret' (password, which should be set at okerrclient to update indicators)
  • Autocreate is on.
Create passive indicator (created from okerrclient) with this policy, e.g.:
okerrupdate --secret MySecret --policy test:passive test:ppp OK
