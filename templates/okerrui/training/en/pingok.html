Lets see if Google is still up?
  • Create indicator (e.g. test:ping1), and set checkmethod 'Ping'.
  • Set 'host' parameter to value 8.8.8.8 . (This is address of Google Public DNS server )
  • Make sure indicator is not in maintenance mode.
  • Wait until it will get status 'OK'. To make things faster, click 'Apply and retest ASAP'
  • When done, click 'Check' on training page to complete task and move to next task.
